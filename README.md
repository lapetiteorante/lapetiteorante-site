# La Petite Orante Catholique

✨ Méditations & Prières quotidiennes ✨  
Méditation de l’Évangile du jour et des lectures de la messe catholique.  
Comprendre les lectures pour vivre leurs enseignements.

---

## À propos
Ce dépôt contient le site statique **La Petite Orante** :  
- `index.html` : page d’accueil affichant les derniers articles.  
- `posts.json` : liste des articles avec titre, date et extrait.  
- `posts/` : dossier contenant les méditations quotidiennes en HTML.  
- `images/` : dossier pour les illustrations.

---

## Objectif
Partager chaque jour une méditation fidèle à la tradition catholique, accessible à tous, pour nourrir la prière et la vie spirituelle.

---

## Déploiement
Le site peut être publié via :
- **Netlify** (recommandé pour la gestion du domaine et HTTPS automatique).  
- **GitHub Pages** (nécessite un fichier `CNAME` avec `www.lapetiteorante.com`).
